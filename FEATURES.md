## User Accounts
- Users can sign up and login
- Each user has a personal profile
- Profile shows their backend work and projects

## Project Showcase
- User can create multiple backend projects
- Each project has:-
    - Name
    - Description
    - Tech Stack
- Projects act like a portfolio section

## API Endpoint Listing
- Each project contains multiple API endpoints
- For every endpoint, users can show:
    - What id does
    - Input (request)
    - Output (response)

## Live API testing
- Anyone can try thr API directly
- They can send requests and see responses

## Execution History
- Every API test is recorded
- Shows:
    - Success/failure
    - Response time
    - When it was tested

## Public Shareable Profiles
- Each user gets a public page
- Each project has a shareable link

## Built in API playground
- Users can modify request data before testing
- Try different inputs and see outputs

## Visibility Control
- Public (visible to everyone
- Private (only for owner)

## Basic Insights
- Show simple stats like:
    - How often APIs are used
    - Success rate
    - Performance trends
